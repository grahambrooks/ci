Welcome to ci

ci is a command line tool that displays the status of your jenkins or hudson builds.

Building

ci is written in C++11 and depends on libcurl and boost. If you are using homebrew you will need to build boost with the --with-c++11 option. I have had a few problems with this on some machines so a compiled binary can be found here: https://github.com/grahambrooks/ci/releases/download/0.0.1/ci-install.dmg
